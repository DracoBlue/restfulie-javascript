h1. How to install

Esta é a versão do Restfulie for javascript

Para executar os testes coloque o projeto no tomcat e abra:
http://localhost:8080/restfulie/restfulie/spec/dom.html

Para usar o restfulie você necessita importar as bibliotecas que 
constam na pasta \webContent\restfulie\lib.

A atual versão suporta:
	resource = Restfulie.at(url).accepts('application/json').get();
	resource = Restfulie.at(url).as('application/json').post(resource);
	
	resource.response.body = conteudo da resposta http (xhr.responseText)
	resource.response.code = codigo da resposta http (xhr.status)
